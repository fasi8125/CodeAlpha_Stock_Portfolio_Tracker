# CodeAlpha Stock Portfolio Tracker

## Overview

A Python application that helps users calculate the total value of their stock portfolio based on predefined stock prices and user-entered quantities.

## Features

- User inputs stock names
- User enters stock quantities
- Calculates total investment value
- Saves result into a text file
- Simple and easy-to-use interface

## Technologies Used

- Python
- Dictionary Data Structure
- File Handling

## How to Run

1. Clone the repository
2. Open terminal

